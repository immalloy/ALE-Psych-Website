Follow these requirements before building ALE Psych from source.

## Requirements

1. Install the latest stable [Haxe](https://haxe.org/download/).
2. Install [Git](https://git-scm.com/downloads) with default options.
3. Run `setup/install-haxelibs.bat` to install required libraries.
4. Install [Visual Studio 2022](https://visualstudio.microsoft.com/es/downloads/).

## Building

Compile for a platform with Lime:

```
lime test [platform]
```
